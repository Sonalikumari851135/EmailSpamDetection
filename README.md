# Email Spam Detection System

## Project Overview

This project is developed to classify emails as Spam or Not Spam using Machine Learning techniques. The system analyzes email text and predicts whether an email is spam or genuine.

## Features

* Email classification into Spam and Not Spam
* Text preprocessing and cleaning
* Feature extraction using CountVectorizer
* Machine Learning model using Naive Bayes
* Model performance evaluation using accuracy score

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* NLTK

## Machine Learning Algorithm

* Multinomial Naive Bayes

## Project Workflow

1. Collected email dataset.
2. Performed data preprocessing and text cleaning.
3. Converted text data into numerical format using CountVectorizer.
4. Split dataset into training and testing data.
5. Trained the model using Naive Bayes algorithm.
6. Evaluated model performance and generated predictions.
